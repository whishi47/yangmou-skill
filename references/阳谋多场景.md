# 商业阳谋场景映射与询问库（七领域并重）

本文件把阳谋原型映射到常见商业情境（销售/营销/管理/职场/投资/产品/谈判），并提供"四轮询问"提问库、六层分析（含现代领域翻译）与方案输出模板。SKILL.md 的多轮询问与六层分析都从这里取料。

> 七领域并重，无默认优先级。用户首轮确认身处哪个领域（销售/营销/管理/职场/投资/产品/谈判），后续提问与打法即按该领域展开；古案经现代迁移可适配全部领域——见 `references/现代领域迁移.md`。当用户的"局"不在销售时，按本文末"多领域情境映射"切换对手画像与筹码的问法即可。

## 一、销售情境 → 阳谋原型（检索命中后对照）

| 销售情境 | 优先命中的支柱 | 推荐原型 |
|----------|----------------|----------|
| 难缠/强势客户压价 | 人性 / 规则 | 丁元英格律诗局、杯酒释兵权 |
| 客户犹豫不决、内部多决策人 | 规则 | 推恩令（下沉决策）、合纵连横 |
| 客户嫌贵、比价 | 规则 / 人性 | Costco透明、论持久战 |
| 客户被竞品锁定 | 规则 / 大势 | 围魏救赵、特斯拉公开专利、华为鸿蒙 |
| 需快速引流/获客 | 人性 | 冯亭献上党（免费钩子）、二桃杀三士（稀缺） |
| 客户不信任行业/你 | 人性 | 朵薇透明生产、特斯拉公开专利 |
| 要重建行业规则/定位 | 规则 / 大势 | 网飞规则重构、Costco会员制 |
| 客户内部有反对派 | 人性 / 规则 | 杯酒释兵权、金刀计、合纵连横 |

## 二、四轮询问提问库（逐轮锁定"局"）

每轮从下面挑 2~4 个最相关的问题问用户，**一轮只问一类，等用户答完再进下一轮**。不要一次全抛。

> **领域切换提示**：若用户的"局"明确属于营销/管理/职场/投资/产品/谈判，把第 2 轮"对手画像"和第 3 轮"我的筹码"按该领域替换（见第五节"多领域情境映射"的对手/筹码定义）。

### 第 1 轮 · 场景定位（你是谁、在哪玩）
- 你卖的是什么？ToB 还是 ToC？客单价大概什么量级？
- 你在这单里扮演什么角色？（销售/老板/渠道/顾问）
- 现在卡在哪一步？（初次接触 / 谈方案 / 谈价格 / 招投标 / 复购）
- 这是单点成交，还是想建立长期锁客/渠道？

### 第 2 轮 · 对手画像（局中人是谁、要什么）
- 真正的决策人是谁？还有哪些人能影响结果？（列出 2~4 个角色）
- 客户/对手最核心的诉求是什么？（省钱、省心、面子、避险、政绩、KPI）
- 他们最怕什么？（被坑、显无能、错过、被上级问责）
- 竞品或内部反对派是谁？他们的软肋/傲慢点是什么？

### 第 3 轮 · 我的筹码（你能公开什么、规则谁定）
- 你手里真正能打的牌有哪些？（价格空间、独家资源、数据、案例、关系）
- 这局里"规则/标准/评判权"在谁手上？你能不能影响或重定？
- 你有什么是敢"亮明牌"的？（敢公开的方法论、成本、结果）
- 你相比对手的不可逆优势是什么？（真本事，阳谋的底座）

### 第 4 轮 · 约束红线（代价与底线）
- 时间窗多长？（本周要签 / 本季 / 长期培育）
- 你最多能让出什么？绝对不能碰的底线是什么？
- 如果对方反制（压价、拖、找竞品），你能承受到第几层？
- 这单失败了，对你是否有连带风险？（声誉/现金流）

## 三、六层语义分析（拿到检索结果后逐层推）

1. **解构**：列出局中人（你、决策人、影响者、竞品/反对派），各自真实诉求与恐惧。
2. **锁死**：判断靠哪根绳（规则/人性/大势）能把对方的最优解绑成"对你有利"。这是阳谋成立的关键，必须在方案里点明。
3. **匹配**：用 `retrieve.py` 命中的原型，选 1~2 个最贴合的，说明可迁移点（参考 `cases.json` 的 `transfer_models`）。
4. **【新增】现代领域翻译**：把命中的古案/商业原型，用 `references/现代领域迁移.md` 第三节（营销心理学对照）或第四节（销售定价模式对照）翻译成目标领域的现代打法。即：古案锁死机制 → 现代语汇（锚定/稀缺/标准/透明…）→ 领域落地动作。若用户要的是非销售领域，此步必做。
5. **明牌**：设计你要"公开"的动作，以及要构建的"势"（信任/标准/紧迫/赛道）。引用 marketing-skills / sales-playbook 的具体玩法。
6. **推演**：预判对方会怎么反制（压价、拖、找竞品、内部搅局），逐条给防御动作。

## 五、多领域情境映射（非销售局怎么问）

当用户说的"局"落在以下领域，第 2 轮"对手画像"与第 3 轮"我的筹码"按此替换：

| 领域 | 谁是"对手" | 第2轮怎么问（诉求/恐惧） | 第3轮筹码侧重 |
|---|---|---|---|
| **营销** | 用户注意力 + 竞品 | 目标用户痛点？他们在怕什么/渴望什么？竞品占着哪块心智？ | 内容资产、数据、KOL、稀缺资源、透明素材 |
| **管理** | 部门墙 / 元老阻力 | 阻力部门的核心 KPI 是什么？最怕失去什么权力/资源？ | 可分配的利益、战略话语权、组织规则制定权 |
| **职场** | 派系 / 上司偏见 | 关键人是谁？他最在意政绩/面子/安稳中的哪个？ | 业绩证据、可借的"战略名义"、公开挑战的底气 |
| **投资** | 标的风险 / LP信心 / 竞买方 | 标的最怕什么（估值崩/政策变）？LP 怕什么（回撤/错失）？ | 信息差、低价筹码、组财团能力、退出通道 |
| **产品** | 用户习惯 / 竞品生态 | 用户现在用什么？迁移成本在哪？竞品生态壁垒多高？ | 规则重构能力、开放标准、切换成本设计 |
| **谈判** | 谈判对方 | 对方底线在哪？最怕失去哪张牌（时间/资源/面子）？ | 必救点、后手反制、利益交换空间 |

各领域通用的阳谋提示见 `references/现代领域迁移.md` 第六节。

## 六、落地的禁忌（务必守住）

- 不教用户骗客户、造假数据、埋隐性条款——那是阴谋，破坏复购与声誉。阳谋靠真"势"。
- 二桃杀三士、金刀计这类"制造内耗/猜疑"的原型，只用于激发健康紧迫感或离间竞品，**绝不用于搞垮客户/同事自己的团队**。
- 阳谋前提是"真有势"：没产品/价值底气，亮明牌会反噬。方案里若发现用户底子虚，先建议补内功。
