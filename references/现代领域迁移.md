# 阳谋·现代领域迁移手册

> 本文件回答一个核心问题：**古案是古代的，怎么"换思考"用到今天的销售、营销、管理、职场、投资、产品、谈判？**
> 答案：古案不可直接搬，但**锁死机制（规则/人性/大势）是可迁移资产**。迁移 = 抽三支柱 → 翻成现代语汇 → 套领域框架。

---

## 一、为什么"纯灌原书 ≠ 能用"

《孙子兵法》《鬼谷子》原书是春秋战国的语境：诸侯、兵法、君臣。直接照搬"推恩令""杯酒释兵权"到今天的 ToB 销售，语境对不上。

但每个古案都有一根**锁死对手的"绳"**，这根绳只有三种材质：
- **规则**（制度/标准/评判权）——对手按规则走最优解 = 对你有利
- **人性**（荣誉/贪婪/恐惧/懒惰/从众）——对手被动机驱动，自己走进死局
- **大势**（趋势/技术代差/时间窗）——对手逆势即输，明知也破不了

**迁移的本质：抽出这根绳的"材质与编法"，用现代语汇重新编一遍，套进现代领域。** 这就是 `cases.json` 里 `transfer_models`（通用思维模型）和 `fields`（各领域打法）两个字段的由来。

---

## 二、三支柱的现代翻译

| 古支柱 | 现代语汇（可直接替换的"绳子材质"） | 阳谋要点 |
|---|---|---|
| **规则** | 合同条款、算法规则、平台标准、定价结构、验收标准、SOP、默认选项（Choice Architecture） | 让对手的"合规最优解"恰好 = 对你有利；你定义规则 = 你收税 |
| **人性** | 锚定、稀缺、社会认同、损失厌恶、禀赋效应、互惠、承诺一致、权威偏见、默认效应、对比/诱饵（详见第三节 70+ 心理模型） | 利用动机但光明正大——不骗、不埋雷，靠"势"不靠"诈" |
| **大势** | 行业演进、技术周期、政策风向、网络效应、复利、时间窗、关键规模（Tipping Point） | 借不可逆的潮流，对方逆势即输；用时间换空间 |

## 三、营销心理学 ↔ 阳谋原型对照

来源：`~/.workbuddy/skills/marketing-skills/references/marketing-psychology/SKILL.md`（70+ 模型精选）

| 心理模型（现代） | 对应阳谋原型 | 现代打法（阳谋式·光明正大） |
|---|---|---|
| 稀缺 / 紧急（Scarcity/Urgency） | 二桃杀三士 | 限时名额、专属权益制造"不争就亏"，让决策人争先 |
| 锚定效应（Anchoring） | 挟天子以令诸侯 | 先亮高价/竞品差距数据，框定对方预期，后续报价显合理 |
| 社会认同 / 从众（Bandwagon/Social Proof） | 挟天子（权威变体：标杆） | 客户数、案例墙、热销标识——"大家都在用"即合法性 |
| 损失厌恶（Loss Aversion） | 论持久战 / 围魏救赵 | "不行动就亏"框架，比"你能得到"更有力 |
| 禀赋效应 / 零价效应（Endowment/Zero-Price） | 冯亭献上党 / Costco | 免费试用、会员"拥有感"，用户难放弃 |
| 互惠（Reciprocity） | 杯酒释兵权（给好处） | 先给价值（内容/工具），再要承诺，对方自然回报 |
| 承诺一致（Commitment&Consistency） | 合纵连横（绑定） | 小承诺→大承诺，让用户/伙伴一步步锁进来 |
| 默认效应 / 选择架构（Default/Nudge） | 网飞规则重构 | 预设最优选项，用户"懒得改"即选你 |
| 对比 / 诱饵（Contrast/Decoy） | 二桃 / 分层打包 | 三档定价中，用诱饵让目标档成显然选择 |
| 权威偏见（Authority Bias） | 挟天子 | 专家、KOL、认证、媒体背书，反对你=反权威 |

> 用法：在做"人性"支柱的阳谋时，先查上表，再从 marketing-skills 调对应模型的完整玩法（如 Hick's Law 防决策瘫痪、BJ Fogg 行为模型设计行动门槛）。

## 四、销售定价模式 ↔ 阳谋原型对照

来源：`~/.workbuddy/skills/sales-playbook/pricing_pattern_library.py`（10 个模式）

| 定价模式（现代） | 对应阳谋原型 | 说明 |
|---|---|---|
| anchor-with-data（数据锚定） | 挟天子 + 围魏救赵 | 先亮竞品差距数据，紧迫感自己卖，不靠嘴说 |
| tiered-packaging（三层打包） | 二桃杀三士（对比选项） | 锚定高价，落地中间档，约 55% 选中间 |
| competitive-ego-trigger（竞争 ego 触发） | 丁元英借傲慢 / 围魏 | "对手#1 你#14"触发 CEO 竞争心，诱其回应 |
| strategic-involvement-upsell（战略参与升级） | 推恩令（下沉价值层） | 同团队不同价值层，价格可差 3-5x |
| bridge-offer（桥接 offer） | 论持久战（时间窗） | 内部招聘要 6-9 月，我搭桥立刻启动 |
| performance-skin-in-game（绩效对赌） | 杯酒释兵权（利益交换） | 低底价 + 对赌奖金，绑定利益，化解"太贵" |
| value-math-on-screen（屏上算 ROI） | 挟天子（权威数据） | 关键词价值可视化，客户自己说服自己 |
| compound-effect-close（复利打包） | 网飞规则重构 / 推恩令 | 多服务复利 4x，防客户只买单项 |
| reference-customer-drop（标杆客户） | 挟天子（权威） | 具体可验证结果 + 牵线，信任爆炸 |
| in-house-team-framing（内部团队框架） | 推恩令 / 杯酒释兵权 | 重构比较基准：从"vendor vs vendor"到"自建 team vs 即用 team" |

> 用法：在做"销售"场景阳谋时，先查上表，再从 sales-playbook 调具体话术（脚本自带 example_dialogue）。

## 五、迁移工作表（5 步，逐层做）

1. **抽锁死机制**：这条古案靠哪根绳把对手绑死？（规则 / 人性 / 大势，可多选）
2. **抽象思维模型**：写成一句通用原理（直接参考 `cases.json` 的 `transfer_models` 字段）
3. **选目标领域**：销售 / 营销 / 管理 / 职场 / 投资 / 产品 / 谈判（可多领域）
4. **套现代框架**：查第三节（人性）或第四节（销售），引用 marketing-skills / sales-playbook 的具体玩法
5. **出明牌方案**：设计公开动作 + 造势（信任/标准/紧迫/赛道）+ 反制推演（见 SKILL.md 第 6-7 步）

示例（推恩令 → 职场）：
> ① 锁死=规则（利益下沉）② 模型="让每个分支自愿受益，反对者的子单元反过来帮你" ③ 领域=职场 ④ 框架=把总目标拆成各部门主管都受益的子目标 ⑤ 明牌="把跨部门项目 KPI 设计成双赢指标，公开承诺资源倾斜"，反制=若某主管仍阻挠，用其下属的支持反压。

## 六、各领域"阳谋思维"速查

- **销售**：对手是客户/竞品。用定价模式（第四节）+ 围魏/挟天子/杯酒。目标：成交且长期锁客。
- **营销**：对手是用户注意力+竞品。用心理学模型（第三节）+ 二桃/论持久战/Costco。目标：心智占领+裂变。
- **管理**：对手是部门墙/元老阻力。用推恩令/杯酒/合纵。目标：让阻力变助力，不硬碰。
- **职场**：对手是派系/上司偏见。用挟天子（借战略名义）/金刀计（谨慎）/丁元英（公开挑战）。目标：上位且不被反噬。
- **投资**：对手是标的风险/LP 信心/竞买方。用论持久战/冯亭（钩子）/合纵（组财团）。目标：低价入局+高价退出。
- **产品**：对手是用户习惯/竞品生态。用网飞（规则重构）/华为（切换成本）/特斯拉（开放标准）。目标：定义赛道。
- **谈判**：对手是谈判对方。用围魏（攻必救）/丁元英（诱先出价）/杯酒（利益交换）。目标：让对方主动让步。

## 七、禁忌（迁移时务必守住）

- **不教骗术**：不造假数据、不埋隐性条款、不误导——那是阴谋，破坏复购与声誉。阳谋靠真"势"。
- **不制造互害**：金刀计、二桃杀三士这类"内耗/猜疑"原型，只用于激发健康紧迫感或离间竞品，**绝不用于搞垮客户/同事自己的团队**。
- **底气优先**：若产品/价值站不住，先补内功再亮明牌，否则反噬。
